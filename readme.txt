		НАСТРОЙКИ в файле config.json:
"api_id": api_id полученный на сайте телеграмма
"api_hash": api_hash полученный на сайте телеграмма
"phone": телефон который использовался при регистрации
"group_link" - полная ссылка на группу в формате "username"
"TIMEOUT" - Время,одидания между отправой сообщениями которые присылаются в группу 


		ЗАПУСК 
Перейдите в командной строке в папку с ботом

Чтобы установить все библиотеки необходимые для работы введите в командной строке - "pip install -r requirements.txt"

Чтобы запустить бота выполните команду - "python3 main.py"
